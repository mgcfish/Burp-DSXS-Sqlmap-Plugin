DSXS Integration for Burp
=========================

DSXS from Here: [DSXS GitHub](https://github.com/stamparm/DSXS)

Installation
------------------------

1. Drop burp pro in ./lib
2. ant clean; ant compile; ant jar;
3. launch burp with the plugin jar in the classpath
